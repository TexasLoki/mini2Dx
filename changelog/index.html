Changelog

Below are the changes that have occurred between each version of mini2Dx.

Version 0.9.2

  • [Enhancement] Desktop dependency injection now uses Reflections library
  • [Fix] #17 Fixed restart method in Animation class not working
  • [Fix] Fixed edge case in position tracking within RegionQuad

Version 0.9.1

  • [Enchancement] #5 Added tile property support to Tiled classes
  • [Enchancement] Improved Quad/RegionQuad implementations
  • [Enchancement] Added volume control to CrossFadingMusicLoop
  • [Enchancement] Added rotation methods to Rectangle class
  • [Enchancement] Added rotation methods to Animation class

Version 0.9.0

  • [Feature] Added Entity-Component-System framework via mini2Dx-ecs library
  • [Feature] Added CrossFadingMusicLoop
  • [Feature] Added RegionalQuadTree for building optimised collision maps
  • [Feature] Added annotation-based dependency injection
  • [Enhancement] API now enforces proper timestep implementation
  • [Enhancement] Improved geometry classes (Point, Line, LineSegment, Rectangle, Shape)
  • [Enhancement] Added pre/post transition in/out methods
  • [Enhancement] Added getFrame(int index) to Animation
  • [Enhancement] Rectangles now support rotation
  • [Enhancement] Added clipping support to Graphics
  • [Fix] Fixed lastGid calculation in Tileset
  • [Fix] Fixed edge case when scaling and rotation of Graphics are combined

Version 0.8

  • Initial release
  • Basic and screen-based game support
  • Basic transition effects
  • Singleton dependency injection
  • Orthogonal Tiled Map Support
  • Maven support
